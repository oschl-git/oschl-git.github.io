coding game

Java 17 | 2022

CodingGame is a school project I've worked on in 2022. It's a simple game in which the player controls a character with code.

There are 5 commands available: walk, shoot, turn_left, turn_right and wait. You can also specify a number in brackets to repeat a command that amount of times.

There are 7 levels in the game.

There is no runnable version of this game available, you can however download the source code on GitHub and compile it yourself. It uses Java 17.

Visit the GitHub page: https://github.com/oschl-git/CodingGame
